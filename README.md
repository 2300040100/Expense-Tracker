# Expense Tracker 💰

A full stack expense tracking web application with Telegram Bot integration.

## 🌐 Live Demo
[https://expense-tracker-taz0.onrender.com](https://expense-tracker-taz0.onrender.com)

## Tech Stack
- **Backend:** Python, Flask, SQLAlchemy
- **Database:** SQLite
- **Frontend:** HTML, CSS, Bootstrap, Chart.js
- **Bot:** Telegram Bot API
- **Tools:** Git, GitHub, Render

## Features
- User registration and login with secure password hashing
- Add, edit, delete expenses
- Dashboard with doughnut and bar charts
- Telegram Bot — just send "spent 150 on lunch" to log expenses
- Auto category detection (Food, Transport, Shopping etc.)

## Telegram Bot Usage
1. Search your bot on Telegram
2. Send `/link your@email.com`
3. Send `spent 150 on lunch`
4. Check dashboard — expense logged automatically!
